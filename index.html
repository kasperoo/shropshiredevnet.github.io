Upcoming Speakers

Marcus Robinson

How far can developers go it alone with DevOps?

Wednesday 22nd February 2017

During this session we will try and answer the question of is it possible for developers to successfully do DevOps without any dedicated operations resource?

We will discuss:

  • - How DevOps and DevOps practices can enable efficient software delivery.
  • - How the adoption of public cloud has reduced the reliance on infrastructure professionals.
  • - How the emergence of PaaS and Serverless offerings is changing the responsibilities of many developers.

To reinforce the discussions DevOps practices will be demonstrated using technologies such as Microsoft Azure App Services, Visual Studio Team Services and Azure Application Insights.

Mike Taulty

Developing with HoloLens: The Path to Mixed Reality

Thursday 27th April 2017

Microsoft HoloLens is the first fully self-contained holographic computer running Windows 10.

It is completely self-contained-no wires, phones, or connection to a PC needed. HoloLens allows you to place holograms in your physical environment and provides a new way to see your world.

This session provides a developer’s overview of the HoloLens and holographic app development in Unity and C#.
